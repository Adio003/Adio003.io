Foire Aux Questions

Comment configurer le formulaire de contact ?

Inscrivez-vous sur Formspree.io avec votre e-mail (adioblw54@gmail.com). Créez un formulaire, récupérez l’endpoint (ex. https://formspree.io/f/xxxxxxxx), et mettez-le dans l’attribut action du formulaire. Testez en envoyant un message.

Puis-je ajouter d’autres champs au formulaire ?

Oui, nous avons ajouté un champ "Service" (liste déroulante). Pour d’autres champs (ex. téléphone), modifiez le HTML avec un nouvel <input> et configurez Formspree pour accepter les données supplémentaires.

Comment personnaliser le style du formulaire ?

Le formulaire utilise des boutons verts pour uniformité. Pour changer les couleurs, modifiez les classes Tailwind (ex. bg-green-500 à bg-blue-500) ou ajoutez du CSS personnalisé dans la balise <style>.

Dois-je garder WhatsApp et le formulaire ?

Oui, les deux sont complémentaires. WhatsApp est rapide pour les questions courtes, le formulaire est idéal pour les demandes détaillées (ex. devis). Vous pouvez aussi activer Tawk.to pour un chat en direct.

Comment activer Tawk.to ?

Inscrivez-vous sur Tawk.to, créez un widget, et remplacez le script placeholder dans <head> par votre script personnalisé. Le chat apparaîtra en bas à droite.

Comment payer via FedaPay ?

Cliquez sur "Payer" pour un service (ex. Netflix), suivez les instructions FedaPay, et entrez vos informations de paiement. Après succès, vous serez redirigé vers une page de confirmation.

Quels sont les délais pour les services ?

Les abonnements streaming sont activés sous 24h, les paiements de factures sous 12h, et les services informatiques (ex. déblocage) dépendent du diagnostic (contactez-nous pour un devis).

Puis-je vous contacter sans WhatsApp ?

Oui, utilisez le formulaire de contact ou envoyez un e-mail à adioblw54@gmail.com. Tawk.to (chat en direct) sera bientôt disponible.

Comment héberger mon site ?

Déployez sur Netlify ou Vercel en important votre code depuis GitHub. Achetez un domaine (ex. via Namecheap) pour une URL personnalisée comme bolanletechs.com.
